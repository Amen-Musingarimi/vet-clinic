## 📖 VET CLINIC <a name="about-project"></a>

In this project, we utilize a relational database to construct the primary data framework for a veterinary clinic. Our approach involves designing a table to house data pertaining to animals, populating it with some sample data, and running queries to retrieve relevant information.

## 💻 Getting Started <a name="getting-started"></a>

This repository includes files with plain SQL that can be used to recreate a database:

- Use [schema.sql](./schema.sql) to create all tables.
- Use [data.sql](./data.sql) to populate tables with sample data.
- Check [queries.sql](./queries.sql) for examples of queries that can be run on a newly created database.

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- To install [PostgreSQL](https://www.postgresql.org/)

### Setup

Clone this repository to your desired folder:

```sh
  cd vet-clinic
  git clone https://github.com/Amen-Musingarimi/vet-clinic.git
```
